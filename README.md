## Development

### Build

|Name|Command|
|:---|---|
|Build (Prod)|`yarn build` or `yarn build-prod`|
|Build (Dev)|`yarn build-dev`|
|Generate Documentation|`yarn build-docs`|

Building dev will bump the build number. E.g. from `1.0.0-22` to `1.0.0-23`
